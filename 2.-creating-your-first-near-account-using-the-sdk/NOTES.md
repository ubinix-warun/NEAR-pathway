# 2. Create your first NEAR account with the NEAR SDK


```
nvm use v12.18.3
npm init -y

npm install --save near-api-js dotenv js-sha256

uuidgen
xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx

node create_account.js

```
